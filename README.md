# Herausgeber - Система публикации новостей

## Описание
Herausgeber - программа на Ruby для автоматической публикации новостей на WordPress и другие платформы.

## Требования
- Ruby 3.0 или выше
- Гем `rubypress` (устанавливается через bundle install)
- Доступ к WordPress сайту через XML-RPC

## Установка

1. Установите зависимости:
```bash
bundle install
```

2. Настройте файл `config.yml`:
```yaml
wordpress:
  host: "https://yoursite.com/xmlrpc.php"
  username: "your_username"
  password: "your_password"
```

**Важно:** Не коммитьте файл config.yml в репозиторий!

## Структура папки с контентом

Папка должна содержать:
- **Один** файл изображения с `_logo.` в имени (например, `news_logo.jpg`) - будет использован как обложка
- **Один** текстовый файл `.txt` - содержит текст новости
- Дополнительные изображения (.jpg, .png, .gif, .webp) - будут вставлены в тело поста
- Видеофайлы (.mp4, .mov, .avi, .wmv) - будут вставлены в тело поста

### Формат текстового файла (*.txt)
```
Заголовок новости

Первый абзац текста новости.

Второй абзац текста новости.

Третий абзац и так далее...
```

- Первая строка - заголовок новости
- Последующие строки - абзацы текста (пустые строки игнорируются)

## Использование

### Запуск с указанием пути к папке с контентом:
```bash
ruby herausgeber.rb /path/to/content_folder
```

### Запуск из папки с контентом (использует текущую директорию):
```bash
cd /path/to/content_folder
ruby ../herausgeber.rb
```

## Форматирование контента

Программа автоматически формирует контент в формате Gutenberg для WordPress:

- **Абзацы**: Стилизованные блоки с отступами и классом `ajustify indent`
- **Изображения**: В рамке (#6bcae1, 2px, radius 8px)
- **Видео**: В рамке (#a54200, 3px, radius 5px), первое видео с autoplay
- **Футер**: Разделитель и ссылка на канал MAX

## Обработка ошибок

Программа проверяет:
- Наличие единственного файла-обложки с `_logo.` в имени
- Наличие единственного текстового файла *.txt
- Корректность файла конфигурации
- Подключение к WordPress

При ошибках выводится подробное сообщение.
